variables

A program that assigns a number in a variable and prints it, then assigns another number in the same variable and prints it.

برنامج يقوم بتعيين رقم في متغير وطباعته، ثم تعيين رقم آخر في نفس المتغير وطباعته.

Copied 
    #include<stdio.h>

    int main() {
      int a = 5;
      printf("%d\n",a);
      a = 7;
      printf("%d\n",a);

    }

A program assigns a number in a variable, prints it, and then assigns another variable with the value of the first variable, and it must be incremented by 5

يقوم البرنامج بتعيين رقم في متغير وطباعته ثم تخصيص متغير آخر بقيمة المتغير الأول ويجب زيادته بمقدار 5

Copied 
    #include<stdio.h>

    int main() {
      int a = 5;
      printf("%d\n",a);
      int b = a+5;
      printf("%d\n",b);

    }

Amusement program : A new amusement park has been established with a strange system.. Each person pays for the entrance ticket according to his age.. If the person is 10 years old, he pays 10 pounds, and so on.. Ahmed, Sherif and Dina went to the amusement park and wanted to know the total amount they would pay Required: Write a program that records the age of 3 people in variables and prints the amount they will pay in amusement parks on the screen.

البرنامج الترفيهي : تم إنشاء مدينة ملاهي جديدة بنظام عجيب .. كل شخص يدفع تذكرة الدخول حسب عمره .. إذا كان عمر الشخص 10 سنوات يدفع 10 جنيهات وهكذا .. أحمد، شريف و دينا ذهبوا إلى مدينة الملاهي وأرادو معرفة إجمالي المبلغ الذي سيدفعونه المطلوب: كتابة برنامج يسجل عمر 3 أشخاص في متغيرات ويطبع المبلغ الذي سيدفعونه في مدينة الملاهي على الشاشة.

Copied 
    #include<stdio.h>

    int main() {
      int Ahmed = 15;
      int Sherif = 14;
      int Dina = 11;
      printf("the amount = %d\n",Ahmed+Sherif+Dina);

      }

Write a program that assgins a person’s age in a variable (Example: 23) and assgins the year in another variable (Example: 2022), then the program subtracts them and prints the result on the screen (Example: 1999).

اكتب برنامجا يحدد عمر الشخص في متغير (مثال: 23) ويحسب السنة في متغير آخر (مثال: 2022)، ثم يقوم البرنامج بطرحهم وطباعة النتيجة على الشاشة (مثال: 1999).

Copied 
    #include<stdio.h>

    int main() {
      int age = 23;
      int year = 2022;
      printf("%d\n",year-age);

    }

Now you know how to assigns a number in a variable, but let's do this better. Required: Write a program that assigns a number in one variable and another number in another variable and prints the two variables between them with a + sign, then print their sum at the end. Example x + y = 15.

الآن أنت تعرف كيفية تعيين رقم في متغير، ولكن دعونا نفعل ذلك بشكل أفضل. المطلوب: كتابة برنامج يخصص رقماً في متغير واحد ورقما آخر في متغير آخر ويطبع المتغيرين بينهما بعلامة + ثم يطبع مجموعهما في النهاية. مثال س + ص = 15.

Copied 
    #include<stdio.h>

    int main() {
      int x = 15;
      int y = 30;
      printf("x + y = %d\n",x+y);

    }

Areas program: Sharif wanted to make some modifications in his room and wanted to determine the area of the room. Required: Make a program that assigns two variables, the first for the width of the room and the second for its length, multiplying the two numbers to calculate the area and then printing the result at the end.

برنامج المناطق: أراد شريف إجراء بعض التعديلات في غرفته وأراد تحديد مساحة الغرفة. المطلوب: عمل برنامج يخصص متغيرين الأول لعرض الغرفة والثاني لطولها وضرب الرقمين لحساب المساحة ثم طباعة النتيجة في النهاية.

Copied 
    #include<stdio.h>

    int main() {
      int width = 320;
      int length = 200;
      int area = width*length;
      printf("%d\n",area);

    }

Program to convert meters to centimeters. Required: Write a program that assigns a number in a variable, then converts this number to centimeters and prints it.

برنامج تحويل الأمتار إلى سنتيمترات. المطلوب: كتابة برنامج يقوم بتعيين رقم في متغير، ثم تحويل هذا الرقم إلى سنتيمترات وطباعته.

Copied 
    #include<stdio.h>

    int main() {
      int m = 5;

    }
