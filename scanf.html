Scanf

A program that takes 4 numbers from the user and combines the first two numbers, then subtracts the third number, then multiplies it by the fourth number and prints the result.

برنامج يأخذ من المستخدم 4 أرقام ويجمع الرقمين الأولين ثم يطرح الرقم الثالث ثم يضربه في الرقم الرابع ويطبع النتيجة.

Copied 
    #include<stdio.h>

    int main() {
      int a,b,c,d;
      printf("please, enter four numbers :\n");
      scanf("%d%d%d%d",&a,&b,&c,&d);
      printf("a = %d\n",a);
      printf("b = %d\n",b);
      printf("c = %d\n",c);
      printf("d = %d\n",d);
      printf("((a + b)-c)×d = %d",((a+b)-c)*d);

    }

A program that asks a person about his age in years and then prints his age in days without leap years.

برنامج يسأل الشخص عن عمره بالسنوات ثم يطبع عمره بالأيام بدون سنوات كبيسة.

Copied 
    #include<stdio.h>

    int main() {
      int a;
      printf("please, enter your age :\n");
      scanf("%d",&a);
      printf("Your age in days : %d\n",a*365);

    }

Program that converts weight from kg to ton (ton = 1000 kg)

برنامج يحول الوزن من كجم إلى طن (الطن = 1000 كجم)

Copied 
    #include<stdio.h>

    int main() {
      float a;
      printf("Please enter the weight in kilograms :\n");
      scanf("%f",&a);
      printf("Weight in tons : %0.1f\n",a/1000);

    }

Each of us dreams of owning many luxurious cars. Required: A program that asks the user how many cars he owns now and how many cars he wants, then prints the rest of the cars for him.

يحلم كل منا بامتلاك العديد من السيارات الفاخرة. المطلوب: برنامج يسأل المستخدم عن عدد السيارات التي يملكها الآن وكم سيارة يريد، ثم يطبع له باقي السيارات.

Copied 
    #include<stdio.h>

    int main() {
      int a,b;
      printf("How many cars do you own now : \n");
      scanf("%d",&a);
      printf("How many cars do you want to have with you :\n");
      scanf("%d",&b);
      printf("The rest of the cars = %d\n",b-a);

    }

There are a lot of students studying graphs in school and we would like to help them. Required: Write a program that takes the values of 3 variables x, y and z and then prints the x + z and the y + z in parentheses eg 5 7 8 (15, 13)

هناك الكثير من الطلاب الذين يدرسون الرسوم البيانية في المدرسة ونود مساعدتهم. المطلوب: كتابة برنامج يأخذ قيم 3 متغيرات x و y و z ثم يطبع x + z و y + z بين قوسين، على سبيل المثال 5 7 8 (15، 13)

Copied 
    #include<stdio.h>

    int main() {
      int x,y,z;
      printf("please, enter a 3 numbers :\n");
      scanf("%d",&x);
      scanf("%d",&y);
      scanf("%d",&z);
      printf("(%d , %d)\n",x+z,z+y);
    }
